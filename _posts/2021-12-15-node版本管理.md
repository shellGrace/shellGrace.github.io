**查看当前 node 版本：**
```tsx
node -v
```
**全局安装：**
```tsx
npm install -g n
```
**查看已经安装的版本列表：**
```tsx
n ls
```
**切换版本：**
```tsx
n 列出所有版本，上下健回车选择所需版本
```
**用指定的版本来执行：**
```tsx
 n use x.x.x【文件名】
```
**删除指定版本：**
```tsx
n rm x.x.x
```
**下载最新版：**
```tsx
n latest
```
**下载指定版：**
```tsx
n【版本号】
```
